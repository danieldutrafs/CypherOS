#  CYPHEROS: Nossa Missão é Acabar com a Procrastinação!

##  Sobre o Projeto

**O CypherOS é um App de Estudos que evolui para ser um Assistente Pessoal de produtividade. Nosso objetivo é transformar a rotina de estudos, fornecendo um caminho claro e adaptativo.**

Acreditamos que a chave para isso é construir uma **Inteligência Artificial (IA)** que realmente personalize a rotina. É por isso que estamos focando na fundação para criar o assistente que nós sempre precisamos.

---

##  Licença e Estratégia Open Core

Para atrair a comunidade e proteger nosso diferencial, adotamos o modelo **Open Core**:

* **Framework (Base):** O código da nossa estrutura (V1) é Open Source, licenciado sob **GNU AGPLv3**.
* **Núcleo de Inteligência:** O código do **Algoritmo de Calibração (V2) e da IA (V4)** será mantido como **Proprietário**. Esta é a nossa fórmula secreta e o nosso diferencial competitivo.

---

##  Construindo a Base (V1)

Nosso objetivo inicial é construir uma base sólida em **MVC** que seja capaz de coletar os dados essenciais para o futuro algoritmo.

* **Ferramentas:** Java, com dependências gerenciadas via **Maven/Gradle** e dados persistidos em **SQLite** (local).
* **Primeira Meta:** Implementar totalmente o **Model** e o **DAO** para as classes principais: **`Tarefa`**, **`Projeto`**, **`Compromisso`** e **`Anotação`**.

---

## Nossa Visão de Futuro

O CypherOS não para no MVP. Este é o nosso plano para as próximas versões:

1.  **V2: O Algoritmo de Calibração**
    * Este é o **coração** do projeto. Iremos programar a lógica para analisar o **Tempo Estimado** vs. o **Tempo Real Gasto**. O sistema aprenderá o padrão de procrastinação individual e fará sugestões de tempo realistas.
2.  **V3: Conexão Social**
    * Iremos adicionar recursos sociais, focando em permitir que os usuários **compartilhem anotações, status de projetos e desafios** para criar uma rede de apoio mútua.
3.  **V4: A CypherIA**
    * Usar os dados coletados para implementar nossa IA, que guiará o aprendizado, recomendando materiais e ajustando planos de estudo em tempo real.
